Exemplos de projeto

Projeto 1: Contratos Inteligentes com Hardhat

Este projeto demonstra como utilizar Node.js em conjunto com o Hardhat para desenvolver, compilar, testar e implantar contratos inteligentes escritos em Solidity. Ideal para quem deseja iniciar no ecossistema Web3 e entender o fluxo completo de desenvolvimento de smart contracts.

Projeto 2: Corretora de Criptomoedas com DREX

Este projeto é uma corretora digital desenvolvida para operar sobre a blockchain do DREX (Real Digital), permitindo a negociação de tokens e NFTs com foco em segurança e escalabilidade. Possui funcionalidades completas de cadastro de usuários, autenticação, e operações de compra e venda, tudo integrado a uma interface amigável e responsiva.
